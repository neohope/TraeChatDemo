## 🚀 快速开始开发

### 环境准备
```bash
# 安装依赖
flutter pub get

# 代码生成
flutter packages pub run build_runner build

# 运行应用
flutter run -d chrome
```

### 开发规范
1. 遵循 Flutter 官方代码规范
2. 使用 Provider 进行状态管理
3. 所有网络请求通过 Repository 层
4. UI 组件保持无状态或最小状态
5. 添加适当的错误处理和日志

### 提交规范
- feat: 新功能
- fix: 修复bug
- docs: 文档更新
- style: 代码格式调整
- refactor: 代码重构
- test: 测试相关
- chore: 构建过程或辅助工具的变动
